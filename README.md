# Cbuilder

## c and c++ project builder
### LibStr.h and file_handle.h are in strlibc

```
<usage> : ./cbuild   <Options> 

Options:
--new <project_name>      create new project

--run <project_name>      run the executable in project

--c                       C project

--cpp                     C++ project

--debug                   make with -g 

```
